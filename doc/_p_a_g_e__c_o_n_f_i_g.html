Параметры пакета конфигурации.

Конфигурация содержит следующие поля:

int safe

Температурный монитор. Указывает номер датчика для контроля:

  • 0 - выключен;
  • 1 - температура CPU (ядра);
  • 2 - температура PCB (печатной платы);
  • 3..6 - температура с внешних проводных датчиков;
  • 7 - средняя температура всех внешних проводных датчиков.
Заметки:
Функционал температурного монитора не реализован.

short t_min

Нижняя граница контроля в градусах Цельсия, 1/16 градуса.
При снижении температуры на контролирующем датчике ниже этого значения, табло выключается с периодической индикацией сообщения "COLD".
Рекомендуется задавать не ниже -40C.
Пример расчета для температуры "-40C": t_min = -40 * 16 = -640.

Заметки:
Функционал температурного монитора не реализован.

short t_max

Верхняя граница контроля в градусах Цельсия, 1/16 градуса.
При превышении температуры на контролирующем датчике выше этого значения, табло выключается с периодической индикацией сообщения "HOT".
Рекомендуется задавать не выше +85C.
Пример расчета для температуры "+85C": t_max = 85 * 16 = 1360.

Заметки:
Функционал температурного монитора не реализован.

int to_watchdog

Сторожевой таймер в секундах. Если в течении установленного времени табло не произведет обмен с управляющим ПК, то табло автоматически перезагрузится.

Предупреждения:
При значениях менее 600 секунд сторожевой таймер не активен!

DWORD autobright

Автоматическое управление яркостью.
Позволяет отключать управление яркостью табло по данным со встроенного датчика освещенности. См. eAUTOBRIGHT.

DWORD to_timer_siren

Таймаут сирены для таймеров в миллисекундах.
