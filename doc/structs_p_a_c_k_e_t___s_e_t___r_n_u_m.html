Структура sPACKET_SET_RNUM

Установка переменных RNUM. Ответ на команду установки переменных RNUM передается в пакете sPACKET_ANS_RNUM. Подробнее...

Поля данных

sMAC_ADR mac
 MAC-адрес контроллера
BYTE cmd
 команда ETH_SET_RNUM.
BYTE type
 тип устройства ETH_TABLO_TYPE.
DWORD p_id
 идентификатор пакета (устанавливается ПК)
WORD index [RNUM_PROTO_NUM]
 индексы чисел в пакете [RNUM_PROTO_NUM]. Если индекс равен 0xFFFF, то элемент игнорируется.
DWORD rnum [RNUM_PROTO_NUM]
 данные чисел в пакете [RNUM_PROTO_NUM].
