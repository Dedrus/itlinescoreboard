Структура sPACKET_SET_RSTR

Установка переменных RSTR. Ответ на команду установки переменных RSTR передается в пакете sPACKET_ANS_RSTR. Подробнее...

Поля данных

sMAC_ADR mac
 MAC-адрес контроллера
BYTE cmd
 команда ETH_SET_RSTR.
BYTE type
 тип устройства ETH_TABLO_TYPE.
DWORD p_id
 идентификатор пакета (устанавливается ПК)
BYTE index [RSTR_PROTO_NUM]
 индексы строк в пакете [RSTR_PROTO_NUM]. Если индекс равен 0xFF, то элемент игнорируется.
BYTE profile [RSTR_PROTO_NUM]
 профили строк в пакете eRPF_STR [RSTR_PROTO_NUM].
BYTE rstr [RSTR_PROTO_NUM][RSTR_LEN]
 данные строк в пакете [RSTR_PROTO_NUM][RSTR_LEN].
