Структура sPACKET_SET_NVSTR

Установка переменных NVSTR. Ответ на команду установки переменных NVSTR передается в пакете sPACKET_ANS_NVSTR. Подробнее...

Поля данных

sMAC_ADR mac
 MAC-адрес контроллера
BYTE cmd
 команда ETH_SET_NVSTR.
BYTE type
 тип устройства ETH_TABLO_TYPE.
DWORD p_id
 идентификатор пакета (устанавливается ПК)
BYTE index [NVSTR_PROTO_NUM]
 индексы строк в пакете [NVSTR_PROTO_NUM]. Если индекс равен 0xFF, то элемент игнорируется.
BYTE profile [NVSTR_PROTO_NUM]
 профили строк в пакете eRPF_STR [NVSTR_PROTO_NUM].
BYTE nvstr [NVSTR_PROTO_NUM][NVSTR_LEN]
 данные строк в пакете [NVSTR_PROTO_NUM][NVSTR_LEN].
