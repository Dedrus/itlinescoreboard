Структура sPACKET_GET_CSTR

Запрос переменных CSTR. Ответ на команду запроса переменных CSTR передается в пакете sPACKET_ANS_CSTR. Подробнее...

Поля данных

sMAC_ADR mac
 MAC-адрес контроллера
BYTE cmd
 команда ETH_GET_CSTR.
BYTE type
 тип устройства ETH_TABLO_TYPE.
DWORD p_id
 идентификатор пакета (устанавливается ПК)
BYTE index [CSTR_PROTO_NUM]
 индексы строк в пакете [CSTR_PROTO_NUM]. Если индекс равен 0xFF, то элемент игнорируется.
