Поля данных

Структура sPACKET_GET_CNUM

Запрос переменных CNUM. Ответ на команду запроса переменных CNUM передается в пакете sPACKET_ANS_CNUM. Подробнее...

Поля данных

sMAC_ADR mac
 MAC-адрес контроллера
BYTE cmd
 команда ETH_GET_CNUM.
BYTE type
 тип устройства ETH_TABLO_TYPE.
DWORD p_id
 идентификатор пакета (устанавливается ПК)
WORD index [CNUM_PROTO_NUM]
 индексы чисел в пакете [CNUM_PROTO_NUM]. Если индекс равен 0xFFFF, то элемент игнорируется.

Подробное описание
